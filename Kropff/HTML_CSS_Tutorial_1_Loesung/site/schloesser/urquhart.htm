Geschichte

  • 1230 - Alan Durward wird Lord von Urquhart
  • 1275 - König Edward I. von England nimmt die Burg in Besitz
  • 1297 - Die Burg wird von den Schotten zurückgewonnen
  • 1303 - Die Burg wird von König Edward I. von England erobert dann wieder von Robert the Bruce (der aus Braveheart)
  • 14. Jhdt - 16. Jhdt. - Die Burg wechselt häufig den Besitzer, da verschiedene Clans darum kämpfen, insbesondere die MacDonalds
  • 1692 - Die letzte Besatzung sucht das Weite, nachdem die Burg nahezu vollständig zerstört worden ist
